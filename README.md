#  AI-Powered Forensic Honeypot (Shield)
**Analyst:** Sanya Asati | MIT-WPU Pune

###  Overview
A high-interaction honeypot designed to capture, encrypt (AES-256), and analyze network threats using a Random Forest Machine Learning model.

###  Key Features
* **Role-Based Access Control (RBAC):** Separate Admin and User portals.
* **AI Inference Engine:** Real-time threat classification using Random Forest.
* **Forensic Integrity:** Automated AES-256 encryption for all captured telemetry.
* **Deception Layer:** Integrated E-commerce "Lure" to capture credential harvesters.

###  Tech Stack
* **Language:** Python
* **ML Library:** Scikit-Learn (Random Forest)
* **Frontend:** Streamlit
* **Database:** SQLite3
* **Security:** Cryptography (Fernet)
